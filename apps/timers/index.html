  Timers

Simple and clean app with support for multiple countdown timers.

You receive notifications when the timers run up.

The app doesn't need to be running to receive the notifications. You'll get them even if the app is closed, or in the background.

After the timers run down, they start counting up, so you know how much time has elapsed since they went off.

That's it. Simple.
